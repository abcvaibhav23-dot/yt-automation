# Generic Prompt Templates

## Template 1: Funny Relatable Shorts
Write a high-retention YouTube Shorts script optimized for watch time and engagement.

Duration: 30-75 seconds
Audience: General audience (all age groups)
Style: Funny, relatable, slightly exaggerated storytelling
Tone: Light, playful, fast-paced

Requirements:
- Start with a hilarious 3-second hook.
- Use short, punchy sentences.
- Add natural pauses using "..." for comic timing.
- Include relatable everyday situations.
- Add one unexpected twist.
- Keep pacing fast and energetic.
- End with a funny or ironic closing line + CTA.

Structure:
1. Funny hook
2. Relatable situation
3. Escalation
4. Twist
5. Punchline ending

Add scene-by-scene visual cues in brackets for stock footage search.
Example:
[Person waking up late in panic]
[Phone showing 15 missed alarms]
[Overdramatic slow-motion reaction]
[Friends laughing]

Add background music suggestion at start:
[Upbeat playful background music]

Optimize for:
- Replay value
- Shareability
- Quick laughs
- Smooth AI voice delivery

## Template 2: Desi Intense Dramatic Shorts
Write a high-retention YouTube Shorts script optimized for watch time and engagement.

Duration: 30-75 seconds
Audience: General audience
Style: Desi, intense, dramatic storytelling
Tone: Bold, powerful, slightly dark humor
Language: Hindi or Hinglish with strong punchy lines

Requirements:
- Start with a strong, powerful hook in first 3 seconds.
- Use short, impactful sentences.
- Add natural pauses using "..." for dramatic effect.
- Build tension quickly.
- Include strong dialogue-style lines.
- Add one unexpected bold twist.
- End with a powerful closing statement.

Structure:
1. Hard-hitting hook
2. Setup with tension
3. Build intensity
4. Strong dialogue moment
5. Bold closing line

Add scene-by-scene visual cues in brackets.
Example:
[Close-up intense stare]
[Slow motion walking in narrow street]
[Moody lighting, dramatic shadows]
[Close-up of clenched fist]

Add background music suggestion:
[Dark tension-building background score]

Optimize for:
- High retention
- Dramatic pauses for AI voice
- Strong emotional impact
- Viral share potential
